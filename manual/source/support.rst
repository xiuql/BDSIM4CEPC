Trouble Running BDSIM
---------------------

For trouble with installation or running, see the :ref:`Troubleshooting` section. Bugs can be submitted to the issue tracker. Please have a look at the existing `list of open bugs <https://bitbucket.org/stewartboogert/bdsim/issues?status=new&status=open&kind=bug>`_ before submitting a new one.

Feature Request
---------------

Feature requests or proposals can be submitted to the issue tracker. Please have a look at the existing `list of proposals <https://bitbucket.org/stewartboogert/bdsim/issues?status=new&status=open&kind=proposal>`_ before submitting a new one.
